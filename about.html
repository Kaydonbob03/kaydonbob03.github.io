About Me

Technical Journey

My technical journey began with HTML/CSS in grade 9, followed by Python in grade 10. Since then, I've embraced the world of programming through various projects. The emergence of ChatGPT in late 2022 opened new learning opportunities, leading to my first major project working with OpenAI's API.

One of my proudest achievements is creating and maintaining a Discord bot, hosted on a modified Raspberry Pi 3B that I converted into a personal cloud server. This project led to developing my first documentation website at kaydonbot.xyz.

My interest in cybersecurity developed through self-study and mentorship from a fellow student who practiced ethical hacking. This combination of web development and security knowledge has shaped my approach to creating robust, secure applications.

Technical Projects & Infrastructure

One of my notable achievements involves repurposing a Raspberry Pi 3B into a versatile server. I modified the hardware with heat sinks and a cooling fan, transforming it into a reliable platform that runs Ubuntu Server. This setup now hosts my Discord bot and serves as my personal cloud storage solution.

This project led to the development of my first documentation website at kaydonbot.xyz, hosted on GitHub Pages. The experience taught me valuable lessons about server management, documentation, and maintaining live services.

Infrastructure Highlights:
